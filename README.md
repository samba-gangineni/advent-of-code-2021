# AdventOfCode 2021

### Requirements:

- .NET 5.0
- Git

### Usage:

1. Clone the repo

```bash
$ git clone https://github.com/samba-gangineni/advent-of-code-2021.git
```

2. run the program

```bash
$ cd advent-of-code-2021
$ dotnet run --project Solve/Solve.csproj [Day] [Problem]
# [Day]: 1 - 25
# [Prob]: 1 - 2
```
